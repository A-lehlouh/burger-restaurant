#  Burger Restaurant Website

A burger restaurant ordering website built with HTML, CSS, and JavaScript.

##  Features
- Interactive menu with food images
- Checkbox-based order selection
- Discount code system
- Total cost calculation
- Social media links

##  Project Files
| File | Description |
|---|---|
| `burger.html` | Main webpage structure |
| `burger.css` | Styling and layout |
| `burger.js` | Interactivity and total calculation |
| `*.jpg / *.png` | Food and icon images |

##  How to Run
1. Download all files into one folder
2. Open `burger.html` in your browser
